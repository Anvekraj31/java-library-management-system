# Library Management System (Java)

## Overview

A simple Library Management System implemented using Java programming language which helps manage books, users and their interactions with the system. The system allows for issuing and returning books, tracking deadlines, and computing any overdue fines.

## Functionality

- User Registration
- Adding and Listing Books
- Issuing Books and Setting Due Dates
- Returning Books and Computing Fines
- Searching for Books by Title or Author

## Technical Requirements

- Core Java
- Object-Oriented Programming Concepts (Classes and Objects)
- ArrayList Data Structure

## Instructions to Run

Compile using:
javac Main.java Book.java User.java

Run using:
java Main

## Working of the System

- Storing books in an ArrayList
- Setting up a due date when issuing books (defaulting to 7 days)
- Charging fines if returned past the due date (10 Rupees per day)
- Easier searching of books based on user queries

## Lessons Learned from Project

This project highlights fundamental object-oriented programming concepts and emulates practical scenarios like issuing books, due dates, and fining overdue books.
